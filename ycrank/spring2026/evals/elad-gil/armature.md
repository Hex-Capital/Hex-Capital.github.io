# Armature -- Elad Gil Evaluation

Armature is the dimensional signature I see most often in YC Spring batches and rarely back: founders with real shipping history in the exact substrate they're attacking (Louis Scremin led AI Agents & MCP servers at Joko, Theodore Otzenberger spent ~5 years at Palantir) paired with zero disclosed revenue curve in a category where Braintrust just took $80M at an $800M post and ships an MCP server integration. The highest dimension is founder-market fit; the lowest is velocity. That pairing describes a competent infra play arriving into a category whose well-funded incumbents are already inside the perimeter Armature wants to defend.

**Perplexity** (2022) is the load-bearing analogy on the founder-ships axis -- Aravind pinged me on LinkedIn and showed up a week later with the thing we'd discussed built, which is the only signal I trust at pre-seed when revenue is absent. Armature passes the *substrate* half of that test: Scremin actually shipped MCP servers at Joko (Joko is now an Armature investor, which is the founder-graph pattern I usually read as confirmatory), and the self-description "we've deployed MCPs used by millions of customers" is the right kind of claim for this product. Where Armature diverges from the Perplexity rung is that the dossier names no shipped artifacts from the founders independently -- no GitHub presence found for either Otzenberger or Scremin, no side-project MCP eval harness already in the wild, no public iteration cadence. Shipping inside Palantir or Joko is not the same legible signal as shipping between investor meetings; I'd want one of them to send me a working multi-harness trace of Claude Code vs. Codex on a real customer MCP before next Tuesday to upgrade this rung.

The revenue-velocity dimension is where Armature lands closest to the failure mechanism I've named explicitly: "if you're an AI company and you're not seeing explosive growth quickly, something's fundamentally broken" (Tim Ferriss Show #863, April 2026). The dossier discloses a pricing page ($79/$159/mo annual, $99/$199 monthly, 7-day trial) but zero customer logos, zero ARR, zero MoM curve, zero Product Hunt or community metric. Pre-seed norms allow for this, and I'm instructed not to penalize the stage default -- but Armature is selling to AI-product teams who *already buy* observability ($16M ARR at LangSmith per getlatka, Braintrust customers including Notion/Replit/Cloudflare/Ramp/Dropbox per SiliconAngle Feb 2026), which means design-partner velocity is the *easy* signal to disclose if it exists. Silence here, in a category with this much enterprise pull, is the false-signal-vs-just-working test failing on the just-working side.

Market re-opening (mechanism analysis): the catalyst Armature implicitly claims is MCP-plus-agent-harness reaching production usage across Claude Code, Codex, Cursor, Gemini CLI, and ChatGPT in 2024-2025. That is a real 24-month catalyst -- closer to the "post-Whisper medical scribing" window that opened **Abridge** (2023) than to the steady-state TAM math I usually pass on. The mechanism that worries me is that the catalyst opened the *category* for everyone simultaneously, including Braintrust, LangSmith, Arize, and Datadog LLM Observability, three of which raised in the last 14 months specifically to attack this surface. Armature's "multi-harness live agent execution against customer MCPs" framing is a product-scope wedge inside an already-catalyzed market, not a re-opening only Armature can see. Eighteen months out, that wedge either widens into a defensible eval primitive or gets eaten by Braintrust's next release note.

Index position (counterfactual): for Armature to score at the **Stripe**-on-e-commerce rung, MCP would need to durably win as the universal agent-to-product protocol AND Armature would need to sit in the test-run path of every MCP shipper the way Datadog sits in the APM path of every backend. The dossier flags both halves as unresolved -- "category dependency on MCP standard adoption" is named as a key risk, and Braintrust already ships an MCP server integration (Braintrust articles, 2026), meaning Armature does not yet occupy the workflow position uniquely. The score flips a tier only if Armature lands 3+ named Fortune 500 MCP-shipper design partners using it as the default pre-prod check, before LangSmith ships the same surface; absent that, Armature is a feature inside someone else's eval product, not an index.

Single-miracle (single-anchor parallel to **Color Genomics**, 2015): Color worked because we isolated the bet to workflow-on-top-of-commodity-sequencing -- one miracle (sequencing keeps getting cheaper) plus execution. Armature's miracle stack is heavier: MCP has to remain the standard protocol, *and* agent test signal has to be deterministic enough to be commercially useful despite the dossier's own admission of "non-deterministic agent test signal" and "the path that only fails on Opus 4.7," *and* Armature has to out-ship Braintrust/LangSmith on multi-harness coverage. That's a compounded-probability profile, not the clean single-miracle architecture I look for; *Unstated Patterns* says workflow-infrastructure-on-top-of-hard-thing is the pattern I back, and Armature qualifies structurally but the underlying "hard thing" (agent behavior) is itself non-stationary in a way commodity sequencing was not.

Forecast (36 months): Armature is most likely acqui-hired by Braintrust, LangChain, or Datadog for $15-40M as a multi-harness eval module, with Scremin staying on to run MCP-specific tooling -- not a venture outcome at scale, but a reasonable founder outcome given the Palantir/Joko networks already funding the round. The mechanism producing this is the lowest-dimension finding: without a disclosed velocity curve into a category with three well-capitalized incumbents shipping adjacent features, Armature's only durable asset is the cross-customer benchmark dataset, which requires customers Armature doesn't visibly have yet.

The strongest opposing case: MCP becomes the TCP/IP of agent-product communication, Joko + Palantir founder networks convert into a dozen enterprise design partners in the next 12 months, and Armature's multi-harness/multi-model test corpus compounds into the eval default the way Datadog APM compounded against New Relic -- at which point a $4M seed at consensus prices into a workflow-infra layer on a re-opening category is exactly the **Decagon** (2023) template. The bear dismantling: Braintrust already raised $80M at $800M post specifically to be "the observability layer for AI" (SiliconAngle, Feb 2026) and ships an MCP server today; Armature's positioning is a scope claim, not a structural moat, and scope claims lose to $80M of engineering throughput unless the velocity curve is already differentiating Armature on the ground -- which the dossier does not show. The condition that flips my verdict: Armature discloses three named Fortune 500 MCP-shipper logos and >$500K ARR within 12 months, ahead of Braintrust shipping multi-harness live execution. Until then this is a neutral -- founders I respect, category I'm long, position I cannot underwrite.
