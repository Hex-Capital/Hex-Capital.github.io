# Memory Store -- Elad Gil Evaluation

Memory Store is a shippy-founders-in-a-contested-lane company with no curve disclosed -- Diwank Singh Tomer (2013 Thiel Fellow, Julep 6.6K GitHub stars, prior product powering Domino's/Wingstop/Buffalo Wild Wings phone ordering) and Ishita Jindal (ex-product at Jar and Lenskart) are exactly the founder archetype I lean into, but the dossier's silence on Memory Store revenue against Mem0's $24M Series A war chest is the analytical signature that decides this one. Highest dimension: founder shipping history. Lowest: revenue velocity. That pairing tells me this is a Letta-adjacent execution play, not a Harvey-style "just working" curve.

Perplexity (2022) is the load-bearing comp on the dimension Memory Store scores best -- Aravind Srinivas pinged me cold on LinkedIn and showed up a week later with the thing built, and that shipping rhythm was the entire reason I wrote the check before the revenue curve existed. Diwank's Julep at 6.6K stars and 973 forks is a real shipped artifact with a measurable user graph, and the Domino's/Wingstop/BWW phone-ordering deployment is independent evidence he ships things that survive contact with enterprise procurement. Where Memory Store diverges from the Perplexity rung is the demo loop: Aravind iterated the *consumer search product itself* between our conversations, whereas Diwank's most legible recent artifact (Julep) is being wound down on 2025-12-31, so the team is restarting commercial GTM from zero rather than compounding a live user base into Memory Store. Strong on the shipping signal, weaker on the "shipping right now, in the current product" version of it.

Harvey (2023) is the failure mechanism to watch on Memory Store's weakest axis: when an AI company can't show the curve, the curve isn't shy, it's absent -- Harvey's $3B → $5B → $8B in months (TechCrunch, Nov 2025) is what "just working" looks like in dollars, and the inverse pattern is what "something fundamentally broken" looks like (Tim Ferriss #863, April 2026). Memory Store's pricing page lists a single $150/user/month Teams tier with a 30-day money-back guarantee and two coming-soon usage tiers (/fast Mode, Reflect), and the dossier explicitly notes "no public Memory Store revenue, user counts, Product Hunt, Chrome install, or social-follower data found." Nine days post-YC Launch is too early to penalize harshly, but $150/seat is materially above Notion/Slack per-seat economics and there are zero named customer logos -- in the AI cycle, real growth is easy to disclose, so silence reads as absence rather than modesty.

On market re-opening (Dimension 1), MCP adoption across Claude, ChatGPT, Cursor, Raycast, and Linear in 2024-2026 is the *kind* of catalyst Memory Store needs, but compounding over 18 months it doesn't behave like Ukraine-catalyzed defense -- it behaves like a protocol diffusion where every client host (Anthropic, OpenAI, Cursor) has incentive to build native team-memory features and capture the value themselves. The catalyst opens a window of maybe 12-18 months for Memory Store before Anthropic ships a first-party "team memory" primitive that makes the integration layer redundant; this is the Lyft/winner-take-all risk in a different costume, not the Stripe/payments-rails durability case.

Counterfactual on index position (Dimension 4): for Memory Store to flip from contested-share to true index-on-AI-collaboration, the dossier would need to show that one of Claude or ChatGPT had *standardized on Memory Store's schema* rather than merely supporting it as one MCP plugin among 100+. The integration list (Gmail, Slack, Granola, Fathom, Claude, ChatGPT, Cursor, Raycast, Linear, OpenCode) is breadth without depth -- Mem0 ($24M, July 2025), Letta ($10M seed, Sept 2024 from Felicis with Berkeley lineage), and Zep ($1M 2024 revenue per CB Insights) all sit in adjacent positions with equal or better claim to the memory-layer primitive. Stripe sat in *the* path; Memory Store sits in *a* path.

Single-miracle parallel: Decagon (2023) on the customer-support side made one bet -- LLM accuracy at handling tier-1 tickets is good enough, the rest is GTM and product. Memory Store's analogous single-miracle frame is "MCP-as-standard plus team-memory-as-a-distinct-category outlasts host-platform absorption," which is one miracle stated charitably and two miracles stated honestly. The Julep technical lineage means the team isn't taking science risk on memory primitives, so I don't price this as multi-miracle in the fusion/biotech sense -- it's a category-definition miracle plus a distribution miracle, which is closer to Decagon's profile than to a Color Genomics workflow-on-commodity bet.

Forecast, 36 months: Memory Store is acqui-hired by Anthropic, Notion, or Glean for the team and the MCP integration scaffolding at a modest outcome ($30-80M), or muddles to sub-$3M ARR against Mem0/Letta and shutters quietly -- the Julep precedent of a wind-down on 2025-12-31 makes the second path mechanically plausible because these founders have already executed one graceful sunset. The mechanism: without an early enterprise curve, the $150/seat pricing can't justify itself before a foundation-model host ships native shared-memory and collapses the wedge.

Strongest bull case: Memory Store is Decagon (2023) for the team-collaboration tier of AI, the MCP catalyst is real, and Diwank+Ishita's four years co-operating Julep is the exact "operationally-disciplined-CEO with prior shipping history" archetype my High Growth Handbook framework profiles. If I'm wrong about MCP being a protocol diffusion rather than a category re-opening, Memory Store at YC-standard pre-seed valuation is asymmetric upside. What dismantles it: Decagon's 2023 wedge came with immediate enterprise revenue velocity disclosed inside 90 days, and Memory Store's dossier offers none -- not vague-early-traction, not pilot counts, not a single named logo. The flip condition is concrete: if Memory Store discloses $500K+ ARR with three named enterprise logos signed in any 90-day window inside the next six months, I re-underwrite at the next round. Until then it's a Neutral with operator respect, not a check.
