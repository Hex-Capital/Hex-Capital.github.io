# Memory Store -- Vinod Khosla Evaluation

Memory Store is a competent serial team building replicable infrastructure in a crowded category — Diwank Tomer and Ishita Jindal are the asset here, not the wedge. The dossier signature is unambiguous: highest dimension is founder learning velocity (three years inside Julep, 6.6k stars, "src/memory-store" already extracted from the monorepo), and lowest is non-linear technical approach (the dossier itself concedes the underlying pattern is "RAG over synced sources + MCP server" that any competitor can replicate). That pairing tells me what kind of company Memory Store is before I even open the consequence question. It is a craft company, not a black swan.

On non-linear technical approach — my lowest rung for Memory Store — the ladder is brutal. Rocket Lab built the Rutherford engine with electric turbopumps and 3D-printed combustion chambers; that is what a 10x cost-reduction looks like in physical reality. DoorDash, the known exception, sat on the middle rung because food-delivery logistics had no technical novelty but planetary consequence overrode it. The low rung is the AI-copilot-for-accountants pattern I pass on every week: existing LLM APIs glued to a workflow. Memory Store sits between the low rung and DoorDash's middle, leaning toward low — Mem0 has $24M and 50,000+ developers, Letta has 13k+ GitHub stars, Supermemory benchmarks at 85.4% on LongMemEval, and Memory Store has not disclosed a single technical metric to differentiate. The dossier's own competitive section calls commoditization risk "high at the technical layer." That is a confession. There is no Rutherford engine here.

On founder as learning machine — Memory Store's highest rung — the ladder runs from the 19-year-old MIT dropout at Mach Industries building weapons systems against Lockheed, down through Tony Xu's immigrant-outsider velocity at DoorDash, to the scripted-resume founders I dismiss within five minutes. Diwank Tomer (Columbia physics, Head of Engineering at Jar, CEO of Whitehead AI, three years inside Julep agent infrastructure) and Ishita Jindal (Babson MBA, Julep co-founder since 2022) sit between the low rung and DoorDash's middle. They have iterated — Memory Store is literally lifted from `src/memory-store` inside the Julep repo, which means they discovered the wedge through use, not whiteboard. But this is also the exact "domain expert with deep resume" profile I usually deprioritize against naive outsiders with first-principles thinking. The Julep-to-Memory-Store transition reads as pragmatic refactoring, not the irreverent leap I funded in Mach.

On consequence magnitude, OpenAI sits at the top of my ladder because AGI reshapes civilization; DoorDash sits in the middle because food access for hundreds of millions is planetary; the AI-for-accountants pass example sits at the bottom because making one profession 40% faster changes nothing fundamental. Memory Store sits adjacent to the bottom rung. If Memory Store fully works, knowledge workers stop copy-pasting context into Claude and Cursor. That is convenience, not consequence. The world does not visibly change. A "personal memory layer" that auto-syncs Fathom and Slack into MCP — no matter how well-executed — does not collapse a basic human cost structure the way Curai collapses primary care or Commonwealth Fusion collapses energy.

Improbable ambition: Hermeus is Mach 5 passenger flight, Impossible Foods is animal-free protein chemistry, the low rung is "known problem, known approach." Memory Store sits at the low rung — the dossier lists six named competitors (Mem0, Letta, Zep, Supermemory, Cognee, plus LangMem and Hindsight) attempting the same thing. When seven companies have the same idea, the idea is not improbable. On radical affordability, Curai delivers AI primary care at near-zero marginal cost, Impossible Foods bends the protein cost curve, and the low rung is per-seat SaaS premium pricing. Memory Store, with its "individual and team" positioning and inferred per-seat model, lands near the low rung — it captures value through a productivity tier, it does not make any expensive necessity free.

My 36-month forecast: Memory Store reaches $1-4M ARR by mid-2029 and is acqui-hired by Notion, Anthropic, or Mem0 in the $20-80M range, or quietly winds down with the team absorbed into a larger AI-tools incumbent. The mechanism is the dimensional signature: competent founders (high) executing a replicable pattern (low) in a category with a $24M-funded leader and a platform owner (Anthropic) capable of shipping native memory into Claude. The Julep pattern repeats — 6.6k stars and $400k ARR is a respectable craft outcome, not a black swan. Memory Store will trace the same arc.

The strongest case against my pass is the DoorDash override: when consequence magnitude is large enough and the founder has high learning velocity, I have funded technical incrementalism. Could Memory Store be DoorDash? No — DoorDash transformed how hundreds of millions of people eat. Memory Store transforms how a few million knowledge workers prompt their AI. The consequence floor is two orders of magnitude lower, and the override does not trigger. The verifiable condition that flips my verdict: Memory Store demonstrates a defensible cross-tool team-graph effect (e.g., 50+ teams where every member's memory references every other member's, producing measurable retention >90% at 6 months) AND publishes a benchmark beating Supermemory's 85.4% on LongMemEval. Without both, this is a craft company, not a Khosla bet. Pass.
