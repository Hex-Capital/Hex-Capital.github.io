# ReasonBlocks -- Elad Gil Evaluation

Sajeev Magesh and Rohan Vij built ReasonBlocks as runtime middleware that claims 42% accuracy lift and 52% token reduction on SWE-Bench Pro, which is the cleanest part of the dossier — one miracle (mid-run reasoning correction works), no chemistry, no regulatory path, no hardware. The problem is that ReasonBlocks pairs that architecturally tidy single-miracle setup with the weakest possible revenue signal in this batch: the website returned only a page title on 2026-05-27, no customer logos, no MRR, no LOIs. That pairing — clean miracle, zero buying behavior — is what ReasonBlocks actually is: a benchmark-stage middleware bet in a category Mem0 just raised $24M into.

On revenue velocity, the ladder runs **Harvey** (2023) at the top — $3B→$5B→$8B in months, the canonical "just working" curve I paid up for — down through **Perplexity** (2022) where usage was visibly compounding before I wrote the check, down to the pre-revenue science pitches I default-pass on. ReasonBlocks sits at the bottom of that ladder. The dossier discloses one self-reported benchmark on a coding eval and a LinkedIn company page; no MoM growth, no design partners named, no pricing page. "If you're an AI company and you're not seeing explosive growth quickly, something's fundamentally broken" (Tim Ferriss Show #863, April 2026) — and in pre-seed I read silence as a negative signal precisely because real curves are easy to disclose. Mem0 announced a $24M Series A in October 2025 with 14M downloads and 41K GitHub stars; ReasonBlocks's CTO GitHub @Rohan-Vij has 20 followers and 41 total stars across 46 repos. That asymmetry is the dimensional gap.

**Lyft** Series C failed my distribution test because it was structurally #2 in a winner-take-all rideshare market; the failure mechanism was "you can't index on the category, you have to beat the leader." ReasonBlocks shows the runtime-middleware version of that same pattern. Mem0 owns persistent memory, LangSmith/Langfuse own observability, LangChain owns orchestration at a $1.25B valuation, and the foundation-model providers ship native caching and reasoning features that can subsume "mid-run correction" as a feature in an Anthropic SDK release. ReasonBlocks's "private reasoning library that grows with every run" is the only index-like claim — per-customer data accumulation if traces compound — but it's unproven, and the surrounding stack has $149M+ of disclosed competitor capital sitting in adjacent boxes. This is not Stripe-on-e-commerce; this is a feature looking for an owner.

ReasonBlocks's market re-opening case rests on agentic AI growing $5.25B → $7.84B between 2024 and 2025 (newmarketpitch.com), which is real but consensus — every YC batch since W24 has had a dozen agent-infra pitches riding the same number. The specific catalyst I'd want named is something like "Anthropic released tool-use August 2024 and production agents now burn $X in token waste my runtime catches" — ReasonBlocks's dossier doesn't isolate that. Compare to **Anduril** (2022) where Ukraine was a dated, specific Pentagon-software inadequacy revelation; ReasonBlocks's catalyst is "average round size rose to $51M in 2025," which is a fundraising-market observation, not a product-market one. Mechanism over 18 months: as model providers ship native cache and reasoning APIs, the catalyst window narrows for third-party runtime layers, not widens.

On founder shipping rhythm, the counterfactual is what would have to be true to flip the score: I'd need to see Magesh or Vij with a prior shipped product at scale, an open-source project with traction, or — the **Perplexity** (2022) pattern — a between-meetings shipping cadence visible in commits or releases. The dossier gives me Rohan-Vij's top pinned repos: LoconomicFulcrum, Memes4Discord, and remote_keylogger (1 star). Magesh's LinkedIn URL 404'd on direct WebFetch and his GitHub handle was not found. Eleven-year co-founder history "since 2nd grade" is a friendship signal, not a shipping signal. USAMO and Stanford CS are pedigree; pedigree is not the disqualifier, but it's also not what gets a check from me — "has this founder actually shipped products before, not just at a big company, but independently" (Hustle Commons) is the test, and the artifacts named in the dossier don't clear it.

On single-miracle architecture, the parallel that loads is **Color Genomics** (2015) — I structured Color as a workflow/data layer on top of commodity sequencing precisely so the bet wasn't on chemistry. ReasonBlocks has the same architectural cleanliness on paper: assume LLMs work, intervene at the runtime, harvest token-cost and accuracy gains. No new physics, no regulatory path. This is genuinely the strongest dimension for ReasonBlocks. The asterisk is that "single miracle architecturally" is not the same as "single miracle commercially" — even if the SWE-Bench Pro numbers replicate, ReasonBlocks still needs distribution against five named, better-capitalized incumbents, which converts the GTM phase into its own miracle.

My 36-month forecast: ReasonBlocks gets acqui-hired by LangChain, Mem0, or a foundation-model provider for sub-$30M as native runtime/caching features in Anthropic or OpenAI SDKs absorb the mid-run correction use case — no path to $10M independent ARR. The mechanism is the high/low signature: the architecturally clean miracle is exactly the kind of feature that ships into a model API or framework SDK, and without a revenue curve to defend independence, the acquihire becomes the rational exit when the next model release closes the token-waste gap.

The strongest bull case against that forecast is the **Decagon** (2023) parallel — workflow infrastructure in a post-LLM open category, validated single miracle, GTM ahead. If ReasonBlocks's 52% token reduction replicates on customer workloads, the per-call economics are obvious and developer self-serve install ("plugs into existing agent stacks in minutes") could compound the way Mem0 did from open-source seed to 14M downloads in 18 months. Agent infrastructure is in my own *AI Market Clarity* "still wide open" list (blog.eladgil.com, Nov 2025). The bear-side rebuttal is that Decagon had named enterprise logos and a curve when I funded it; ReasonBlocks has a benchmark, a 404'd LinkedIn, and a CTO whose pinned repo is a 1-star keylogger. The verifiable condition that flips my verdict: three named enterprise design partners with paid contracts within 90 days, or a public Anthropic/OpenAI integration. Until one of those lands, I pass — same logic as Lyft Series C, market first, and this is a #2-or-worse position in a category Mem0 and LangChain already own.
