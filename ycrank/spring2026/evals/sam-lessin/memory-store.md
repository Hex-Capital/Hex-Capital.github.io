# Memory Store -- Sam Lessin Evaluation

Memory Store is what I'd call a textbook "perfect for Anthropic" company — Diwank Tomer and Ishita Jindal are demonstrably sane operators (Columbia physics, Babson MBA, three years on Julep AI together, spouses, $400k ARR on the prior product) building a sane feature on top of Anthropic's MCP standard. The signature here is high-on-founders, floor-on-durability. When the highest scoring axis is "competent humans" and the lowest is "can survive an Anthropic memory release," I already know the shape of the bet.

Memory Store on the Consensus Mispricing ladder sits between Solana at four cents — where every smart person told me Ethereum won and the market had priced the bet at zero — and the AI-wrapper bucket I pass on at every demo day, where Adobe is the natural buyer. Memory Store gets pinned far closer to the wrapper rung: Mem0 has already raised $24M with Basis Set and Peak XV, Letta has 13k+ GitHub stars, Zep, Supermemory, Cognee, LangMem, and Hindsight are all live in the same "AI memory" category, and Memory Store was reportedly "voted #1 most promising startup at batch kickoff." That last detail is the tell. When the YC batch itself is excited, the price already reflects the upside. There is no four-cent moment here.

Memory Store on the Structural Durability ladder sits between Metropolis — physical parking assets AI literally cannot replicate — and the LLM-wrapper category Adobe absorbs in eighteen months. Memory Store is MCP-server-plus-RAG-over-Slack-Fathom-Granola. Anthropic owns MCP. Anthropic shipped memory in 2025. Notion, Slack, Linear all expose APIs to anyone, which means Anthropic and OpenAI have both the distribution and the data access to ship the native version of Memory Store as a checkbox. This is the Adobe-for-AI pitch I rant about on every podcast. Memory Store sits one inch above the floor only because the team-shared MCP packaging is a real, if narrow, wedge.

Memory Store on the Primitive Creation ladder sits between Venmo — which created the verb "Venmo me" and a behavior that didn't exist — and "better CRM." The product is a synchronization layer over existing tools using a protocol Anthropic published in November 2024. Nobody is going to say "let me Memory-Store that meeting." It's plumbing for a behavior (talking to LLMs with persistent context) that Anthropic and OpenAI are themselves productizing. Pinned just above the LOW rung.

Memory Store on the Sanity-Insanity ladder is the company's strongest axis. The HIGH rung is Raj at Solana — engineer, not hype-guy, watched through three companies. The LOW rung is matchmaker co-founders from a networking event. Memory Store is closer to the high rung than the low one: Tomer and Jindal co-built Julep to 6.6k stars and $400k ARR, brought Hamada Salhab forward from that team, and the new product is literally extracted from a folder named `src/memory-store` inside the Julep monorepo. Sane founders. Real prior reps. The problem is the other half of the ratio — the idea isn't insane. "Memory for AI agents" is the most consensus pitch in the 2026 batch. Sane founder, sane idea is just a company. I need sane founder, *crazy* idea.

Memory Store on the Capital Discipline ladder I have no real signal on — pre-seed, four people, no burn data — so I default. Gusto is the high rung, Allbirds the low. Memory Store gets the middle.

Memory Store in 36 months: aqui-hired by Anthropic, Notion, or a tier-two AI platform for $20-60M after Anthropic ships native cross-tool memory inside Claude and the team-MCP wedge collapses into a feature. The mechanism is the same one that killed every "Adobe for AI" pitch I've ever heard — the platform owns the distribution and the protocol, and the wrapper becomes a hire.

The bull case that stings: Slack. Slack was the hottest deal in the Valley in 2013 and I paid consensus prices because Stewart Butterfield was creating a genuine new behavioral primitive that replaced email, and Microsoft Teams took years to catch up. Could Memory Store be the Slack of agent memory — team-shared, cross-tool, network-effected? The pitch would be: Mem0 is a developer SDK, Letta is an OSS framework, Zep is a backend graph, and nobody is doing what Memory Store is doing at the team-collaboration layer. Maybe the corpus accumulates, maybe the team-graph compounds, maybe the nine integrations become unreplicable distribution. I don't buy it. Slack replaced a behavior; Memory Store stores context for a behavior the platforms themselves are productizing. Anthropic isn't Microsoft circa 2014 — Anthropic *owns* MCP, Anthropic *is* shipping memory. The Slack exception applied because the incumbent was slow and culturally wrong; here the incumbent is fast and structurally aligned. Pass. The condition that flips me: Memory Store signs five named Fortune-500 team accounts paying $50k+ ARR inside twelve months and shows team-seat memory-query density 10x individual seats — that's evidence the team-graph is a real primitive, not a feature.
