# Interfaze -- Peter Thiel Evaluation

The first thing I notice about Interfaze is a competitive landscape that condemns it before analysis even begins. Firecrawl has 50,000 GitHub stars and 350,000 developers. Apify generates $13.3 million in annual revenue processing 6.8 billion API calls. Google Cloud Vision handles OCR at enterprise scale. Kadoa occupies the financial services niche. And frontier model providers — OpenAI, Anthropic, Google — are adding multimodal capabilities that subsume every task in Interfaze's feature list with each quarterly release. When a dossier effectively contains its own "competitive landscape" slide comparing five players in overlapping space, the diagnosis is already terminal. This is not a conspiracy to change the world. It is a crowded bazaar where vendors shout over each other about marginally different implementations of the same commodity functions.

The claimed secret — that developers should route deterministic tasks to specialized small models rather than expensive general-purpose LLMs — fails the most basic test. Ask any ML engineer whether task-specific models outperform general models on narrow, deterministic tasks, and they will look at you as though you have stated that water is wet. This is not an important truth that few people agree with. It is a technical truism that every practitioner knows and that the market is already acting on. The IEEE paper documents an architecture, but architecture is not a secret — it is an implementation. Interfaze has built a competent orchestration layer atop consensus insight. The monopoly structure that should follow from a genuine secret is entirely absent: no network effects compound with usage, no proprietary technology creates a 10x barrier, and the OpenAI API compatibility that enables adoption simultaneously guarantees that any developer can migrate away in an afternoon. The switching costs are zero by design.

The brand confusion between JigsawStack and Interfaze tells me something deeper about the founders' conviction. A company that cannot commit to its own name is a company still searching for its identity — the hallmark of indefinite thinking. The email domain is jigsawstack.com, the GitHub organization carries the old name, the SDKs are published under the prior brand, and prior traction metrics reference a different entity. This is not a team that sat down with a bold, specific plan for what the world looks like when they succeed and then made irreversible commitments toward that vision. This is a team iterating toward product-market fit, rebranding along the way, hoping the market will tell them what to build. Khemlani's prior exit at Stayr — $1 million to $3 million ARR before an undisclosed transaction — demonstrates competence at building small businesses, not the kind of audacious conviction that builds monopolies. The co-founders share no prehistory before JigsawStack. They are rolling dice.

Let me steelman the case. If you believe that the frontier model providers will remain stubbornly monolithic — that OpenAI and Anthropic will never build task-specific routing into their own platforms — then the orchestration layer becomes structurally necessary middleware. In that world, Interfaze's early integrations (n8n, MCP server, Vercel AI SDK) and consumption-based pricing ($1.50/$3.50 per million tokens versus $15-60 for GPT-4-class models) could compound into a developer standard. Ten million API requests in nine months of beta is not nothing — it demonstrates that real developers are using this in production workflows. And the IEEE publication provides at least a veneer of technical credibility that most YC developer tools lack. If small-model orchestration becomes the dominant architectural pattern and Interfaze captures the routing layer, you could construct a path to a meaningful infrastructure business. But "meaningful infrastructure business" is precisely the problem. Even the optimistic scenario describes a well-run commodity — not the kind of winner-take-all monopoly that justifies a power-law bet. Apify already does $13.3 million in revenue as a scraping platform. The ceiling for an orchestration layer competing against both specialized point solutions and the frontier providers themselves is structurally bounded.

The physical-world moat is nonexistent — this is pure software middleware, the kind of layer that a well-funded team could replicate in months. There is no hardware, no regulatory barrier, no physical-world manipulation creating multi-year replication advantages. The proprietary small models for OCR and speech-to-text are trained on standard architectures against available datasets. The benchmarks are self-reported in a company-authored paper with no independent validation, and the system's performance is partially dependent on whichever LLM the user selects — making it unclear how much value Interfaze's own components contribute. We wanted flying cars. We got another API wrapper.
