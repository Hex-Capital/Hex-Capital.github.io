# Memory Store -- Cyan Banister Evaluation

Diwank Singh Tomer is a 2013 Thiel Fellow who has been building agent memory primitives since Julep in 2022, and Memory Store is the productized second act of that work — which makes Memory Store strongest on founder-lived-experience and weakest on the "magically weird, unsexy, capital-intensive" axis that actually triggers my checks. Memory Store is a domain-native founder pair walking into the most consensus YC-adjacent lane of 2025 -- a Mem0 ($24M raised) and Letta ($10M seed) shaped corridor -- carrying a $150/seat SaaS wrapper. That pairing is the signature.

Anduril-Luckey (2017) is the high rung on lived-experience -- Palmer shipped Oculus before he shipped a Lattice. Memory Store rhymes with that template more than the dossier first suggests: Diwank and Ishita Jindal co-founded Julep in 2022 and shipped persistent memory primitives to 6.6K GitHub stars before pivoting the same technical substrate into a team-shared workspace. The shared mechanism is *founder-of-the-place, not visiting* -- Diwank's prior infra reportedly powers Domino's, Wingstop, and Buffalo Wild Wings phone ordering, which is the most real "stateful-conversation-at-scale" credential I have seen in a YC memory pitch this batch. Where Memory Store diverges from the Anduril rung is that Julep's backend is being shut down 2025-12-31 per its own README, which means the 6.6K-star community demonstrably did not convert to a commercial book of business; lived experience is present, but commercial-translation is empirically unproven on the prior reps.

The consensus YC AI-agent darling is my pass template, and Memory Store has the silhouette of it on the magical-weirdness axis. The failure mechanism on that low rung is that the category is already being priced as if the outcome is certain -- Mem0 raised $24M from Basis Set in October 2025, Letta carries a $70M post-money from Felicis, Zep is already at $1M ARR, and Supermemory and Cognee are circling the same primitive. Memory Store enters this lane with no public funding disclosure beyond YC standard and a generic "memory.store" SEO surface that collides with Redis and GCP Memorystore. That is the exact opposite of "find them before it becomes consensus" -- the consensus arrived twelve months ago, and Memory Store is the n+5 entrant on an existing memory-layer ladder rather than the magically-weird founder nobody else can see yet.

The single-CEO question is the governance landmine. Memory Store lists two named co-founders -- Diwank and Ishita -- with no designated CEO and no tiebreaker disclosed in the dossier. The mitigant is the four-year shared operating history through Julep since 2022, which is the kind of pre-relationship that usually defuses the HQ-Trivia (2017) Yusupov/Kroll collapse pattern. But Memory Store has not done the work in public of naming who breaks ties when the GTM-versus-platform decision splits the room, and at $150/seat with no logos that decision is twelve months away, not twenty-four. I score this near my absent-evidence default, with a modest uplift for the prior co-op reps.

Memory Store is a pure software abstraction sold per seat, which is the wrong end of my capital-intensity ladder. If this dimension were going to flip a tier, Memory Store would need to be ingesting a regulated data corpus -- HIPAA-bound clinical notes, FINRA-bound advisor conversations, ITAR-bound defense comms -- where the compliance perimeter compounds into a moat over eighteen months. Nothing in the dossier suggests that posture; the integrations are Gmail, Slack, Granola, Fathom, Claude, ChatGPT, Cursor, Raycast, Linear, OpenCode -- horizontal productivity, not a regulated vertical. That is the lane where Notion and Glean already live, and where Mem0 and Letta will arrive with bigger war chests.

Physical verifiability is the one place Memory Store quietly clears my bar. The Niantic (~2015) test is whether a normal human can install, use, or photograph the product inside 48 hours -- I played Ingress for three years before John Hanke got my check. Memory Store is a native MCP plugin inside Claude, ChatGPT, Cursor, and Raycast, which means I can sign up, point it at my own Gmail and Granola, and feel whether the "Briefs" living-docs surface actually reduces context-switching in my own workflow tomorrow. That is more tactile than the average B2B-API pitch in this batch, and it is the dimension I would actually go test before I made a final call.

My 36-month forecast on Memory Store: acqui-hired by Notion, Glean, or one of Mem0/Letta into the $15-40M range, or quietly wound down below $5M ARR -- because $150/seat for a horizontal memory layer in a category with $34M+ of better-capitalized direct competition gets compressed to a free feature inside Claude or ChatGPT well before Memory Store builds a defensible seat-level retention curve. The mechanism is straight platform substitution: Memory Store's distribution thesis depends on MCP hosts not shipping native team memory, and Anthropic and OpenAI ship native team memory roughly every quarter now.

The strongest bull case against me is the Niantic parallel I almost wrote off. John Hanke spent a decade on location data at Keyhole and Google Earth before mobile GPS made the primitive pluggable -- Diwank has spent since 2012 on agent infrastructure and since 2022 on memory primitives, and MCP is the 2024-2026 platform shift that makes his primitive pluggable into every client agent at once. If you squint, Memory Store is Hanke-in-2014, not consensus-AI-darling-in-2026. That stings because the founder-prior-art shape is genuinely there. The dossier evidence that wins the argument against the bull case is the Julep shutdown of 2025-12-31 paired with the absence of a single named paying-customer logo at $150/seat, alongside the visible $34M+ already deployed into Mem0 and Letta inside Memory Store's exact lane. The verifiable condition that flips my verdict: a Fortune 500 or named YC-batch logo paying Memory Store $150/seat across 50+ seats inside six months of YC Launch -- if Memory Store can show that, the Julep-to-commercial-translation risk has resolved and I write the check.
