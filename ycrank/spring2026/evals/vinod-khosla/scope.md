# Scope -- Vinod Khosla Evaluation

Anand-Arnaud Pajaniradjane is a solo founder building an analytics layer that watches AI agents pick, call, and stumble inside other people's APIs -- and that single sentence locates Scope precisely where my framework objects most: high enough on "why now" to register, far too low on consequence magnitude to register as a check. The dossier names no incumbent regime that collapses when Scope wins. The closest claim is that 22,775 MCP servers exist by May 2026 (Glama directory) and that agent-mediated product usage is becoming a budget line. That is a "picks-and-shovels on a real wave" pitch. It is not an "$15 trillion of GDP" pitch (Fortune 2026-03-06), which is the frame I score against.

**OpenAI** (2019) sits at the top of my consequence ladder because the success-state was civilizational: a labor economy where machines do 80% of economically valuable work by 2030 (Fortune 2025-07-01). Scope's success-state is that Stripe's docs get rewritten so Claude Code stops calling the wrong endpoint, and Anand emails a dashboard to a DevRel lead at a $200M-ARR API company. That is real. It is also Glean-shaped at best -- enterprise information arbitrage -- and Glean only got my override because Arvind Jain combined ex-Google search depth with Fortune 500 information-asymmetry as the consequence variable. Scope has neither the capital scale nor the consequence scale that justified the Glean exception. The dossier does not name a single regime -- not API documentation, not observability vendors, not DevRel teams -- that disappears if Scope wins. Without that named collapse, the consequence dimension cannot exceed mid-single digits on my 30-point scale.

**Pass example** -- the GPT-4 sales-call summarizer that I dismiss in 30 seconds -- failed my technical-risk dimension because there was no scientific miracle, no falsifiable breakthrough, and any Tier-1 generalist could fund it. That exact failure mechanism appears in Scope's dossier. The product is software that simulates agent interactions and captures tool calls, errors, and reasoning traces (tryscope.app). No physics. No biology. No fleet, no plant, no foundation model training run. The dossier explicitly cites Profound shipping "Agent runs across hundreds of inputs" (Profound product update, 2026) -- which means the technical surface is already commoditizing inside an adjacent category before Scope has named a customer. I do not need to write this check. Sequoia, Greylock, any seed fund with an AI-infra thesis can write this check. Capital depth is not the gate, and that is precisely the signal that tells me I am over-resourced for the deal.

On founder-defies-expert-consensus, I will give Anand-Arnaud his due: Princeton interpretability research for closed-source models plus ML engineering inside GEO/AEO is a non-trivial crossover, and most observability founders cannot read attention patterns. **Glean's** Arvind Jain is the cleanest portfolio parallel -- ex-Google Search built information retrieval for the enterprise -- and that exception cost me a known violation of this dimension to make. But Anand's profile is GEO/AEO-engineer continuing into agent-engineer; it is the same playbook one layer down the stack, not a Levine/Finn jump from academic robotics into a foundation-model company. My TED Talk warned against extrapolating the past; Scope's founder bio reads as careful extrapolation, not defiance.

Counterfactual on founder learning rate: for Scope to flip from a Pass to a Neutral on this axis, the dossier would need to name a prior pivot -- a shipped product Anand killed, a thesis he publicly retracted, a company he sold before this one. The **Slash** (2024) override hinged on the Kanye West pivot being documented and load-bearing. The Scope dossier names a single linear path from Princeton to GEO/AEO to founding Scope in 2025. No exit, no pivot, no public thesis update. The absent-evidence default applies and I do not credit beyond mid-single digits.

Required-capital profile is the cleanest single-anchor parallel to **Commonwealth Fusion** (2019) -- by inversion. I led $115M into fusion because the roadmap required deployment scale no consensus VC would touch. Scope's roadmap, as far as the dossier reveals, requires a solo engineer, some agent integrations against Claude Code / Codex / Cursor, and outbound sales to API vendors via cal.com/anand-scope/demo. Five to ten million dollars to first revenue. F4 Fund and YC have already covered the gap. My check size is wrong for this deal in both directions -- too large to be useful, too small to matter.

Thirty-six month forecast for Scope: acqui-hired by Profound, Datadog, or a frontier model lab (Anthropic, OpenAI) for under $30M, with Anand absorbed into an agent-experience or developer-relations function -- or shut down and the founder back into a senior ML role at a foundation lab. The mechanism is the dossier's own competitive picture: Profound at $1B valuation (Yahoo Finance, Feb 2026) already ships agent runs; Anthropic and OpenAI can extend first-party telemetry into the surface Scope instruments; and a solo founder cannot out-integrate nine funded AEO incumbents plus the agent vendors themselves. The high-on-founder-crossover, low-on-consequence-and-capital signature points to talent acquisition, not category creation.

The bull case worth taking seriously: if agent-mediated software consumption really is the next interface layer, then the proprietary dataset of agent failure modes across Claude Code, Codex, and Cursor becomes a benchmark -- a Glean-shaped information arbitrage on how machines, not humans, evaluate APIs. Anand's interpretability background is the rare prerequisite to actually read agent reasoning rather than count tokens. That is the Glean override pattern, and I held my nose on Arvind Jain's domain-expert profile because the consequence scaled. But Glean had capital depth I was uniquely positioned to provide, a Fortune-500-information-asymmetry consequence figure, and a multi-year head start before incumbents reacted. Scope has a solo founder, a demo-gated landing page, zero named customers in the dossier, and Profound shipping the adjacent feature today. The Glean exception requires the consequence and the capital scale; without both, the founder-crossover signal alone does not clear the bar. I would flip this verdict on one verifiable condition: three signed LOIs with named API or MCP vendors at $100K+ ACV each, plus a second hire with foundation-lab provenance. Until then, this is the pass.
