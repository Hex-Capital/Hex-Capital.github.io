# Memory Store -- Paul Graham Evaluation

Memory Store is Diwank Singh Tomer and Ishita Jindal's second act after Julep — a 6.6K-star open-source agent orchestrator they are shutting down on 2025-12-31 — and that one fact dictates everything about how I read this dossier. The cofounder pairing is strong the way I want it to be strong: four years of shared operating history, real hacker lineage, a 2013 Thiel Fellowship on Diwank's side. The animal-resourcefulness column is where Memory Store goes quiet. There is no cereal-box moment here, no Memory Store-specific stunt. The artifact is Julep, and Julep is being wound down. That gap between "proven technical pair" and "no fresh pre-funding hustle for this company" is Memory Store's signature.

On animal resourcefulness, the **Airbnb** (2009) cereal-box pattern is the high rung — strangers paying $40 for $4 of cereal because no investor would write the check. The mechanism is the same one I described in "Relentlessly Resourceful" (March 2009): founders improvising revenue from nowhere when the institutional path is closed. Memory Store does not give me that. What Diwank and Ishita give me is a prior open-source artifact at 6.6K stars and a prior company powering Domino's and Wingstop phone ordering, which is real but it is *prior*. The dossier contains zero text about what they did between deciding to start Memory Store and the YC Launch on memory.store. No unscalable stunt, no manual onboarding of named teams, no signed LOI with a Fortune-500 customer, no Product Hunt run. Julep had 6.6K stars and still got wound down without a commercial base — which means stars are not the animal test, revenue is, and Memory Store has not yet shown me that animal moment for *this* company.

The lowest rung on the schlep ladder is the polished AI-wrapper from my Pass Example — pure software, no offline grind, no regulatory exposure, nothing unpleasant that other YC applicants would unconsciously route around. Memory Store sits one notch above that floor: maintaining live integrations across Gmail, Slack, Granola, Fathom, Claude, ChatGPT, Cursor, Raycast, Linear, and OpenCode is integration schlep, and integration schlep is a real schlep — APIs break, OAuth scopes get revoked, MCP semantics churn. But it is not the **Coinbase** (2012) compliance grind, and it is not the **DoorDash** (2013) last-mile operational ugliness. It is the kind of schlep a well-funded competitor like Mem0 ($24M) can throw engineers at and close in eighteen months. Memory Store has chosen unpleasant work, but not work that's structurally unpleasant enough to be a moat.

On organic-idea fit, the mechanism that matters: did the founders themselves want this product before building it? Diwank and Ishita spent four years building Julep, an agent orchestrator whose core primitives are persistent memory across sessions. Memory Store is, structurally, the team-facing UX wrapper around the problem they spent the last cycle solving for developers. That is real continuity. It is closer to the **Stripe** (2010) pattern — Collisons hacking commerce through **Auctomatic** (2007) before productizing payments — than to anything synthetic. But Stripe's organic-idea claim was airtight because the Collisons were *the customer*; Memory Store's positioning is for "teams" running on Gmail and Slack, not for the founders' own agent-building workflow, which leaves a small but real distance between the founder pain and the productized form.

The eighteen-month mechanism on hacker genealogy: Diwank's Thiel Fellowship and Julep's 6.6K stars compound into recruiting leverage and MCP-spec credibility that a non-technical pair would have to buy. This is the **Reddit** (2005) first-batch pattern — Swartz had co-authored RSS before any of us had a thesis — and it tends to produce companies that ship the next integration faster than competitors. Memory Store will out-ship Mem0 on MCP surface area in any given quarter. What it will not do, on this dimension alone, is out-distribute Glean's enterprise sales motion. Hacker lineage compounds product velocity, not enterprise GTM.

Cofounder configuration is where Memory Store reads cleanest. Diwank and Ishita co-founded Julep in 2022, which gives them roughly four years together before this batch — comfortably past the "met at a hackathon weeks before applying" red flag, and squarely in the **Stripe** (2010) Collison-brothers / **Reddit** (2005) Huffman-Ohanian dorm-roommate zone where long working relationships precede the company. Ishita's product operating experience at Jar and Lenskart pairs with Diwank's technical lineage in the complementary way "What We Look for in Founders" (October 2010) describes. The remaining two unnamed team members do not concern me at four total.

Memory Store in 36 months reaches roughly $1M-$3M ARR, gets crowded by Mem0 ($24M, YC-backed) on the developer side and by Glean/Notion adding shared-context features on the team side, and either pivots toward a developer-API posture or gets acquired by a productivity incumbent — Linear, Notion, or Granola itself — in the $30M-$80M range. The mechanism producing this is the gap between Memory Store's strongest dimension (proven pair with real hacker lineage) and its weakest (no fresh pre-funding animal moment for this product), which is the profile of a company that ships competently but does not generate the wrong-seeming traction story that produces a Black Swan.

The strongest case against this forecast is my own **Stripe** (2010) exception — I funded the Collisons after Auctomatic even though they had already done YC once, because the prior outcome at ages 19 and 17 outweighed my normal process expectations. Memory Store has the same shape: repeat founders, prior shipped product, organic continuity from the last project. If this exception applies, I should be writing the check. But the Auctomatic exit was $5M at 19/17 — a commercial outcome that validated the pair's ability to convert technical taste into money. Julep is being wound down on 2025-12-31 with 6.6K stars and no public commercial base, which is the opposite signal. The condition that would flip my verdict: twenty named paying teams at the $150/seat price within six months of the YC Launch, or a single Fortune-500 logo using Memory Store as system-of-record memory. Without that, Memory Store is a competent neutral, not a black swan.
