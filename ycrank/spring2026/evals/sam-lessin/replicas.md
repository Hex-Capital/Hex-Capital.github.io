# Replicas -- Sam Lessin Evaluation

Here's the thing about Replicas that jumped out at me immediately: the company is building an orchestration layer on top of AI coding agents made by companies that have every incentive and every capability to absorb this functionality themselves. OpenAI's Codex already runs in a cloud sandbox. Anthropic is shipping cloud execution for Claude Code. This isn't a case where the incumbents *might* build the feature in two years — they're building it right now, today, as a core part of their product roadmap. When I talk about the AI kill test, this is the purest possible failure mode. It's not "Adobe could add this." It's "Anthropic and OpenAI are literally shipping this as we speak." You're not even competing with incumbents in an adjacent market. You're building a convenience layer directly on top of products whose makers will make that convenience native.

The mispricing situation here is inverted. AI developer tools is the single most consensus-hot category in venture capital right now. The dossier lists competitors with over $1.2 billion in combined funding. Devin alone is at a $10.2 billion valuation. Every smart investor I know is trying to get into AI coding infrastructure deals. That means the upside is already fully priced in — and then some. When I backed Solana, every crypto person thought Ethereum had won and SOL was worthless at four cents. That's mispricing. When I looked at Eaze, every top-tier VC was avoiding cannabis. That's mispricing. Replicas is the exact opposite: it's a company in a category where capital is flooding in, where the market has already decided this is huge, and where the question isn't whether AI coding agents matter but who captures the value. The answer to that question is almost certainly the agent providers themselves and the well-funded companies like Devin and Factory that have years of head start and hundreds of millions in the bank.

The bull case, and I want to be honest about it, would go something like this: the coding agent market fragments into five or ten viable agents, no single provider wins, and engineering teams desperately need a neutral orchestration layer to dispatch work across multiple agents from their existing workflows. Like Kubernetes became the neutral container layer despite AWS and Google having their own offerings. If that happens, and if Replicas embeds deeply enough into Slack, Linear, and GitHub workflows to create real switching costs, there's a real business here. The "agent-agnostic" positioning could be genuinely valuable in a fragmented market. But this thesis requires two things I don't believe: first, that the agent providers won't build excellent cloud execution into their own products (they already are), and second, that the workflow integrations themselves constitute a meaningful moat (they don't — any competent team can build a Slack bot and a GitHub integration in a week).

There's no primitive being created here. "Run a coding agent in the cloud instead of locally" is an operational convenience, not a new behavior. Nobody is going to say "the Replicas of X" about anything. Venmo created a verb. Patreon created the behavior of paying creators directly. Replicas removes the need to keep your terminal open while an agent works. That's a feature. The 30%+ PR completion rate they cite is a metric of the underlying agents' capability, not of Replicas' value — if Claude Code gets better at completing tasks, that number goes up regardless of whether it runs locally or in a Replicas VM.

Connor Loi seems like a capable young engineer — Waterloo CS, shipped real code at Helicone, won a hackathon. But this is a solo student founder going up against Cognition ($900M raised), Factory ($50M+), and Augment Code ($252M). And the idea isn't insane — it's completely obvious. Every developer who's used Claude Code has thought "I wish I could just fire this off and come back to a PR." That's the problem with obvious ideas in consensus-hot categories: the insight isn't scarce, the execution window is closing fast, and the well-capitalized incumbents are already executing on it. I need a sane person pitching an insane idea. This is a sensible person pitching a sensible idea in the most crowded market in tech. That's just not how I make money.
