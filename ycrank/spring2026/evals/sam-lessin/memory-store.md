# Memory Store -- Sam Lessin Evaluation

Memory Store is the cleanest possible expression of "technically credible founders, wrong asset class for my check" -- Diwank Singh Tomer is a 2013 Thiel Fellow with fourteen years in AI and a 6.6K-star open-source repo behind him, and he's pointing all of that horsepower at the single most consensus-bid sleeve in the 2026 batch: the AI memory layer, sitting one shelf over from Mem0's fresh $24M Series A from Basis Set and Peak XV. The dimensional signature is high founder, low category, standard cap table -- which is the exact profile that produces respectable acqui-hires and zero fund-returners.

The Pass Example I wrote for the canonical 2026 YC AI-agent pitch reads like it was drafted with Memory Store's YC Launch tab open. Mem0 just cleared $24M, Letta cleared $10M from Felicis on a $70M post out of Berkeley, Zep is already at ~$1M of 2024 revenue with a Graphiti graph that nobody has caught yet -- the "memory for AI" category is not pre-consensus, it is mid-consensus and the price-discovery has already happened one tier upstream. Memory Store enters this with no disclosed round terms, which under my absent-evidence default means YC-standard SAFE at YC-standard cap, which means I'm being asked to pay a 2026 AI-memory price for the fourth-or-fifth visible entrant. Solana at $0.05 worked because every generalist had quit crypto; Memory Store at MCP-plugin-for-teams works only if Mem0, Letta, and Zep all simultaneously fail to ship a team-shared UX layer, and three funded competitors don't usually all whiff in the same window.

Teamshares is the rung I'd want Memory Store to sit near on capital structure -- an employee-ownership rollup nobody else could underwrite because the instrument itself was the alpha -- and instead Memory Store is shipping a $150/user/month SaaS plan with a 30-day refund window and "Reflect" and "/fast Mode" tiers marked coming soon. That is a vanilla per-seat cap table that will need a Series A inside 18 months to keep pace with Mem0's already-raised $24M war chest, then a B to fund the enterprise motion against Glean, then a C to survive whichever MCP host (Anthropic, OpenAI, Cursor) decides team memory is a feature. The $150/seat sticker without a single named customer logo in the dossier is the specific tell -- it's pricing aspiration, not pricing power, and pricing aspiration is what burns the next two financing tacks.

Diwank Singh Tomer being a 2013 Thiel Fellow who has been shipping AI since 2012 is a real signal on conviction -- the kind of founder-input my framework actually rewards -- and Julep crossing 6.6K stars while powering phone ordering for Domino's, Wingstop, and Buffalo Wild Wings is the rare dossier line that survives the "what did you actually build" test. Over 18 months, this conviction compounds if Diwank can publish opinionated long-form on why MCP-as-memory-substrate beats the SDK-for-developers framing Mem0 owns -- I'd want to read three essays from him before the term sheet, and the dossier surfaces none. Ishita Jindal scaling product at Jar and Lenskart to millions of users is the operator complement, and four years of shared operating history with Diwank on Julep clears my co-founder-dynamic red flag cleanly. The conviction is there; what's missing is the public artifact of it.

Counterfactually on founder-as-brand: for Memory Store to flip from low-mid to high on this axis, Diwank's @diwanksingh would need a retrievable follower count north of 20K with consistent posting on agent memory, Julep's 6.6K stars would need to be migrating into a Memory Store Discord with measurable weekly actives, and the YC Launch would need to be the third distribution event not the first. None of that is in the dossier. The Julep wind-down on 2025-12-31 is the part that actually cuts against the founder-brand thesis -- a 6.6K-star community is being told the backend is shutting down at the exact moment the founders need that community to follow them into a paid product. That's not Bookshelved's owned-audience-into-product motion; that's a developer community being asked to swallow a shutdown notice and a $150/seat upsell in the same calendar quarter.

On category sleeve, compare Memory Store to Sublime Security on the single axis of pre-consensus positioning -- Sublime entered enterprise email security when the SOC-SaaS crowd was distracted, and Memory Store is entering AI memory when Mem0, Letta, Zep, Supermemory, and Cognee are all already named on the n1n.ai 2026 comparison post. "AI is just more internet" is exactly the lens here: this is a productivity wrapper on an MCP substrate that Anthropic and OpenAI are both incentivized to absorb natively. The category is not in my convicted sleeves -- it sits directly inside the AI-infra-at-seed-pricing line I cross out on principle.

Thirty-six-month forecast: Memory Store gets acqui-hired by Anthropic, Cursor, or a Glean-tier incumbent for under $50M by Q4 2028, with Diwank and Ishita landing on the acquiring team's memory or context group; the standalone product never crosses $5M ARR because the $150/seat price collapses on first enterprise procurement cycle and the bottoms-up motion gets eaten by whichever MCP host ships native team memory first. The mechanism is simple: founder quality clears the acqui-hire bar, category gravity prevents the venture outcome, cap-table standardness means the exit math doesn't return a seed at any reasonable entry price.

The strongest bull case against this is that MCP is genuinely the platform shift the dossier claims, Memory Store is the only entrant explicitly positioning at the team-collaboration layer rather than the developer-SDK layer Mem0 owns, and Diwank's Julep technical lineage means they ship the shared-context UX before Mem0 figures out it needs one -- in which case $150/seat at 500 teams is $9M ARR and the Series A prints. That case stings because the positioning gap between "memory layer for app builders" (Mem0) and "shared brain for teams" (Memory Store) is real and underexploited. What dismantles it: Glean already sells the team-knowledge motion to the enterprise buyer at higher ACVs with full SSO/SCIM/audit, and Notion will ship MCP-native memory inside 12 months because they have to -- Memory Store is squeezed top and bottom by incumbents who don't need to win the category, only deny it. The single verifiable condition that flips my verdict: ten named paying team logos at the $150/seat price with documented six-month retention by end of Q2 2027, published as case studies on memory.store. Absent that, I'm out at the YC-standard cap, and I'd revisit only on a down round or a structural pivot to a non-SaaS instrument.
